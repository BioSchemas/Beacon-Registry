GA4GH Beacon

 
Information

Select the chromosome, and location of your observed allele, along with the value observed

The server will return "Yes" or "No" to indicate that we have already observed the particular allele (and "error" if there was a problem retrieving data from the data file)

Note: server will respond "No" if the allele is the reference allele

(Maximum of 250000000)

Notes

Programmatic interface

Query Parameters:

ass
Assembly (optional) - currently only "GRCh37";
src
Source of data (optional) - see values above or "all"
chr
Chromosome name - either 1-22 or X/Y/MT
pos
1-based position - a positive integer (less than length of chromosome)
all
Allele - 'A', 'C', 'G' or 'T'

Response:

'Yes'
Allele observed in dataset(s);
'No'
Allele not observed in dataset(s);
'ref'
This is the reference allele at the given location;
'error'
Error retrieving data from data source.
'invalid'
One of the parameters passed is invalid, the location you have supplied is invalid, and/or one of the required parameters is missing

Usage Notes:

  • Minimal URL for requesting: http://www.sanger.ac.uk/sanger/GA4GH_Beacon?chr=?;pos=?;all=?
  • There is no formal limit on the number of requests made to this beacon, but please be considerate to other users of the Beacon and other services hosted at the Wellcome Trust Sanger Institute. The webteam reserve the rights to throttle/block any usage which adversly affects this and other services hosted on the same hardware.
